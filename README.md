🏠 Real Estate Landing Page

A modern and responsive Real Estate Landing Page built using HTML, CSS, and JavaScript.
This project showcases property listings, attractive UI design, and smooth user experience.
---
 🚀 Features:-

* Responsive design (Mobile + Desktop friendly)
* Modern UI/UX layout
* Property showcase section
* Smooth scrolling and navigation
* Clean and well-structured code

---
🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
---
 📂 Project Structure
* index.html
* style.css
* script.js
---
🎯 Purpose of Project
This project is created for learning and showcasing front-end development skills. It can be used as a template for real estate websites.
